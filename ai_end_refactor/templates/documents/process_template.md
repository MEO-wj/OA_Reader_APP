# 内部流程规范模板

请使用此模板创建流程规范文档。

## 文件格式

- 文件名：`{流程名称}.md`
- 编码：UTF-8
- 格式：Markdown

## 字段说明

| 字段名 | 说明 | 示例 | 必填 |
|--------|------|------|------|
| 流程名称 (name) | 流程名称 | 员工入职流程 | 是 |
| 适用范围 (scope) | 适用范围 | 全体员工 | 是 |
| 流程步骤 (steps) | 操作步骤 | 见下方 | 是 |
| 责任人 (owner) | 流程责任人 | 人力资源部 | 是 |
| 生效日期 (effective_date) | 生效日期 | 2026-01-01 | 是 |

## 流程格式示例

```markdown
# {流程名称}

## 基本信息

- 适用范围：{适用范围}
- 责任人：{责任人}
- 生效日期：{生效日期}

## 流程步骤

### 第一步：{步骤名称}
{具体操作说明}

### 第二步：{步骤名称}
{具体操作说明}

## 注意事项

1. {注意事项1}
2. {注意事项2}
```

## 注意事项

1. 文件名建议使用清晰的流程名称
2. 流程步骤尽量详实，包含具体操作
3. 如有表单或附件需求，请单独说明
