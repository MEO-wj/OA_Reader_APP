# 常见问题解答（FAQ）模板

请使用此模板创建 FAQ 文档。

## 模板说明

- 文件名：`faq_{主题}.md`
- 编码：UTF-8
- 格式：Markdown

## 文件格式

```markdown
# {主题}常见问题

## 问题分类

### 分类1：{分类名称}

**Q1：{问题1}**

A1：{答案1}

**Q2：{问题2}**

A2：{答案2}

### 分类2：{分类名称}

**Q3：{问题3}**

A3：{答案3}

---

## 附录

如有其他问题，请联系：{联系方式}
```

## 字段说明

| 字段名 | 说明 | 示例 | 必填 |
|--------|------|------|------|
| 主题 (topic) | FAQ 主题 | 员工福利 | 是 |
| 分类 (category) | 问题分类 | 入职福利、日常福利 | 是 |
| 问题 (question) | 问题描述 | 如何申请年假？ | 是 |
| 答案 (answer) | 解答内容 | 详细说明... | 是 |
| 联系方式 (contact) | 联系方式 | hr@company.com | 否 |

## 注意事项

1. 问题尽量简洁明确
2. 答案尽量详实可操作
3. 定期更新 FAQ 内容
4. 保持分类清晰合理
