# OA-Reader v1.2 → v0.3.x 宣发文案
# 你的校园AI助手，悄悄变强了

还记得第一次打开 OA-Reader 的感觉吗？
也许你只是想查一条通知，却要在学校OA里翻半天。
几个月前我们发布了 v1.2，那时候它还只是一个"能用"的工具。
今天，我们想告诉你：它变强了很多。

---

## 后端重写：快了多少？

说个可能你不在意但我们在意了很久的事：我们把整个后端从 Python 重写成了 Go。

你不需要知道 Go 是什么。你只需要知道——
打开文章列表，快了。搜索通知，快了。AI 回复，也快了。
这是我们送给你的第一份升级。

## AI 记忆系统 v2：它开始记住你了

以前你问完就忘，AI 也忘。

现在不一样了。全新的记忆系统会让 AI 记住你的使用习惯：
你关心的通知类型、你常问的问题、你的偏好。
聊得越多，它越懂你。就像一个真正了解校园事务的学长。

## 搜得更准了：不只是关键词

全新的三层智能检索——

- **向量搜索**：理解你问题的含义，而不是只匹配字面
- **关键词模糊搜索**：即使记不清标题也能找到
- **AI 重排序**：把最相关的结果排在最前面

还支持按日期范围筛选，最近的通知自动优先展示。

## 更多细节升级

- 文章列表现在显示发布时间和附件信息
- 上传头像支持自动压缩，加载更快
- 校园 SSO 一键登录，不用再记密码
- 历史通知补录完成，更早的 OA 也能查到了
